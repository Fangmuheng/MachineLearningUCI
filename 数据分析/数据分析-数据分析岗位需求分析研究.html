对于缺失值的处理方式通常有：
1.删除这些变量是缺失值(unknown)的行；
2.如果预计该变量对于学习模型效果影响不大，可以对unknown值赋众数，这里认为变量都对学习模型有较大影响，不采取此法；
3.可以使用数据完整的行作为训练集，以此来预测缺失值。



但我们只是对数据进行分析，并不对数据进行建模，且缺失值不多，对其他行没有影响。在这里我们直接删掉有缺失值的行即可。

def salary_count(x):
    x_min,x_max = x.split('-')  #获取-两边的最大最小值
    x_min = int(x_min.replace('k','000'))
    x_max = int(x_max.replace('k','000')) #截取字符串，把k改成000，将字符串转化为数字
    return np.mean([x_min,x_max])
